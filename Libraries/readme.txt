@brief 		how to insert a library here into your project
@note 		these are read-only, with full source control!

@section 	Procedure
	Create a 'Lib' Group in Project
	Open Dir of Library
		*e.g. '/Users/justin/Documents/Ref/Libraries/Programmatic UI'
	Drag target files for use under 'Lib/'
		*by reference
	

	Open your target project in xCode
	
	R.C. on Lib/ dir
	Add Files to...
	Select Library target (e.g. Ref/Libraries/Programmatic UI) -> 'Add'
		*Optional, for ref
	Any class you want access to, now simply drag to Lib/
		*e.g. Lib/PrgUI/Test.swift -> Lib/Test.swift
	...
	done!
